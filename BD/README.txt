Para criar as tabelas do projeto numa base de dados de desenvolvimento (máquina pessoal, etc), é preciso se logar na base de dados como SYS ou SYSTEM e criar o usuário que será usado pela aplicação executando o script create_user.sql.

Para criar as tabelas na base de dados da Fatec SO, executa-se o script main.sql com o usuário BD1711006, no servidor apolo:1521/xe, senha occupyMars.
